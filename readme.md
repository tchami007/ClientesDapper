# ClientesDapper

Se trata de una aplicacion de tipo Web API, creada para estudiar la implementacion de apis
usando los siguientes caracteristicas:

- Patron CQRS, usado para separar responsabilidades de commands y queries
- Patron Mediador, con la libreria MediatR
- Implementacion de Clean Arquitectura nivel inicial: Separacion en capas, carpeta domain, inyeccion de dependencias
- Cross cutting : Carpeta "Shared" para librerias y clases comunes
- Dapper como Orm

Aunque es una aplicacion de consultas, sirve como pase para hacer apis por caso de uso
